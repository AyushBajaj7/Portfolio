A modern, interactive portfolio website showcasing my work as a Software Developer specializing in full-stack development, machine learning, and cloud infrastructure.

## Features

- **Scroll-Driven Animation**: 300-frame avatar animation that responds to scroll position
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Dark/Light Theme**: Toggle between color schemes with smooth transitions
- **Interactive UI**: Glass-morphism design with hover effects and micro-interactions
- **Project Showcase**: Detailed cards with tech stack and external links
- **Contact Form**: Easy ways to connect via email and social platforms

## Tech Stack

- **Frontend**: React 19, TypeScript, Tailwind CSS 4
- **Build Tool**: Vite 8
- **Animation**: Framer Motion, Canvas API
- **3D Elements**: React Three Fiber (basic integration)
- **State Management**: Zustand
- **Icons**: Lucide React, Material Symbols

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This site is automatically deployed to GitHub Pages via GitHub Actions workflow on every push to `main`.
